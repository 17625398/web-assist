# 提示词

Page Assist 提供三种类型的提示词来增强您的 AI 交互体验：

1. **自定义提示词** - 在聊天界面中使用
2. **Copilot 提示词** - 内置上下文菜单操作（参见 [Copilot 提示词](/sidebar/copilot.md)）
3. **自定义 Copilot 提示词** - 创建您自己的上下文菜单操作（参见 [自定义 Copilot 提示词](/sidebar/copilot.md#custom-copilot-prompts)）

## 自定义提示词

自定义提示词是您可以在 Page Assist 聊天界面中创建和使用的提示词。这些与出现在浏览器上下文菜单中的 Copilot 提示词不同。

您可以通过进入 `设置` → `管理提示词` → `自定义` 选项卡来创建自定义提示词。

![Custom Prompts](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-19%20205135.png)

有两种类型的自定义提示词：

1. 系统提示词
2. 快速提示词


### 系统提示词

系统提示词将被设置为 LLM 中的 `system` 类型提示词。这意味着该提示词将作为系统提示词发送给 LLM。这对于设置对话的上下文很有用。

#### 支持的系统提示词变量

您可以在系统提示词中使用以下变量：

- `{current_date_time}` - 当前日期和时间（本地格式）
- `{current_year}` - 当前年份
- `{current_month}` - 当前月份（0-11）
- `{current_day}` - 当前月份的日期
- `{current_hour}` - 当前小时（0-23）
- `{current_minute}` - 当前分钟（0-59）

这些变量在提示词发送给 LLM 时会自动替换为相应的值。

### 快速提示词

快速提示词是您可以用来快速向 LLM 发送提示词的提示词。您可以通过点击输入框中的 `快速提示词` 按钮来使用快速提示词。

如果您在 `{}` 括号中放置变量，它们将被自动选择。
