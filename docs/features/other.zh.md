# 其他功能

这些是 Page Assist 提供的小型实用功能。

## 宽屏模式

对于较大的屏幕，您可以从设置中启用宽屏模式。

1. 进入设置
2. 在 `通用设置` 部分下
3. 启用 `启用宽屏模式` 选项

![Wide Mode](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-19%20212707.png)

感谢 [@yz778](https://github.com/yz778) 提供此功能。

## 为之前的聊天恢复上次使用的模型

如标题所述，您可以为之前的聊天恢复上次使用的模型。如果您想在切换到旧聊天时使用相同的模型，这很有用。

1. 进入设置
2. 在 `通用设置` 部分下
3. 启用 `为之前的聊天恢复上次使用的模型` 选项


## 打开 Web UI 时恢复上次聊天

您可以在打开 Web UI 时恢复上次聊天。如果您想从上一次离开的地方继续聊天，这很有用。

1. 进入设置
2. 在 `通用设置` 部分下
3. 启用 `打开 Web UI 时恢复上次聊天` 选项

## 打开侧边栏（Copilot）时恢复上次聊天

您可以在打开侧边栏（Copilot）时恢复上次聊天。如果您想从上一次离开的地方继续聊天，这很有用。

1. 进入设置
2. 在 `通用设置` 部分下
3. 启用 `打开侧边栏（Copilot）时恢复上次聊天` 选项


## 使用 AI 生成标题

标题生成是根据对话为聊天生成标题的功能。如果您想为聊天生成标题，这很有用。

1. 进入设置
2. 在 `通用设置` 部分下
3. 启用 `使用 AI 生成标题` 选项
