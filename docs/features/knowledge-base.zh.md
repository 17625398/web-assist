# 知识库

Page Assist 支持知识库功能，可用于与您自己的数据聊天。您可以使用它来与您自己的数据进行对话。

::: warning
请谨慎使用此功能。由于没有服务器端存储，数据将被处理，嵌入将存储在浏览器存储中。这可能会导致性能问题。
:::

## 支持的文件类型

- PDF
- Docx
- Txt
- CSV
- MD

## 如何使用知识库

要使用知识库，您需要从 RAG 设置中设置一个嵌入模型。我们建议使用 `nomic-embed-text` 或任何支持文本的嵌入模型。不要为此使用文本到文本模型。

1. 进入设置
2. 进入管理知识库
3. ![Knowledge Base](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-19%20210054.png)
4. 您可以通过点击 `添加新知识` 上传您的文件

处理文件需要一些时间。完成后，当您检查输入框时，您会看到一个知识库的块图标。您可以点击它并选择您想要使用的知识库。
