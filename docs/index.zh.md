# 欢迎使用 Page Assist

欢迎使用 Page Assist，您的本地 AI 模型的浏览器伴侣！通过 Page Assist，您的网页浏览体验将进入一个新的智能和高效维度。

## 什么是 Page Assist？

Page Assist 让 AI 交互变得轻松！只需：

- 使用我们时尚的侧边栏从任何网页与您的 AI 聊天
- 访问强大的 Web UI 控制中心
- 连接到您喜爱的本地 AI 模型
