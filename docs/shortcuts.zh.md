# 快捷键

Page Assist 支持以下快捷键：

| 操作         | 快捷键         |
| ------------ | -------------- |
| 打开侧边栏   | `Ctrl+Shift+Y` |
| 打开 Web UI  | `Ctrl+Shift+L` |

您可以在 Chrome 扩展管理页面的扩展设置中更改键盘快捷键。

## 应用快捷键

| 操作     | 快捷键       | 描述                 |
| -------- | ------------ | -------------------- |
| 新聊天   | `Ctrl+Shift+O` | 开始新的聊天对话     |
| 切换侧边栏 | `Ctrl+B`     | 打开/关闭聊天历史侧边栏 |
| 聚焦文本框 | `Shift+Esc`   | 聚焦消息输入字段     |
| 切换聊天模式 | `Ctrl+E`     | 在普通聊天和当前页面聊天之间切换 |

## 更改键盘快捷键（浏览器特定）

要更改键盘快捷键，请按照以下步骤操作：

### Chrome

1. 导航到 `chrome://extensions/shortcuts` 进入扩展设置页面。

2. 找到 Page Assist 扩展。

3. 更改所需操作的快捷键。

_*注意*: 这适用于基于 Chromium 的浏览器，如 Edge、Brave 等。_

### Firefox

1. 导航到 `about:addons` 进入附加组件页面。

2. 点击 `设置` 图标。

3. 点击 `管理扩展快捷键` 按钮。

![Manage Extension Shortcuts](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-15%20114332.png)

4. 找到 Page Assist 扩展。

5. 更改所需操作的快捷键。

_*注意*: 这适用于基于 Firefox 的浏览器，如 Zen、Librewolf 等。_
