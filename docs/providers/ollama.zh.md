# Ollama

Ollama 让您在本地机器上运行大型语言模型。

Page Assist 默认支持 Ollama。如果您在 `localhost:11434` 上使用 Ollama，则不需要配置任何内容。Page Assist 会自动检测到它。

如果您在使用 Ollama 时遇到任何问题，请查看 [Ollama 连接问题](/connection-issue.md) 指南。


## 多个 Ollama 实例

您可以通过以下步骤配置多个 Ollama 实例：


1. 点击浏览器工具栏上的 Page Assist 图标。

2. 点击 `设置` 图标。

3. 进入 `OpenAI 兼容 API` 选项卡。

4. 点击 `添加提供者` 按钮。

5. 从下拉菜单中选择 `Ollama`。

6. 输入 `Ollama URL`。

7. 点击 `保存` 按钮。

您不需要添加任何模型，因为 Page Assist 会自动从您配置的 Ollama 实例中获取它们。
