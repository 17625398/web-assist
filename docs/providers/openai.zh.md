# OpenAI 兼容 API

Page Assist 支持 OpenAI 兼容 API 端点。您可以在 Page Assist 中使用任何 OpenAI 兼容 API 端点。

默认情况下，Page Assist 支持以下 OpenAI 兼容 API 端点：

- LLaMA.cpp
- LM Studio
- Llamafile
- Ollama
- OpenAI
- DeepSeek
- Fireworks
- Novita AI
- Hugging Face
- Groq
- Together
- OpenRouter
- Google AI
- Mistral
- Infinigence AI
- SiliconFlow
- VolcEngine
- TencentCloud
- AlibabaCloud
- vLLM
- Moonshot
- xAI
- Vercel AI Gateway
- Chutes
- Anthropic (Claude)
- CanopyWave
- BigModel (Zhipu)
- MiniMax


## 添加 OpenAI 兼容 API


1. 点击浏览器工具栏上的 Page Assist 图标。

2. 点击 `设置` 图标。

3. 进入 `OpenAI 兼容 API` 选项卡。

4. 点击 `添加提供者` 按钮。

5. 从下拉菜单中选择 API。如果它不在默认列表中，选择 `自定义` 并输入 API URL 和 API 密钥。

6. 如果需要，添加 API 密钥。

7. 点击 `保存` 按钮。


::: info
对于 Ollama、LM Studio 和 Llamafile，您不需要添加任何模型，因为 Page Assist 会自动从 API 中获取它们。
:::
