# LM Studio

您可以将 LMStudio 的 API 添加到 Page Assist。以下是操作步骤：

1. 点击浏览器工具栏上的 Page Assist 图标。

2. 点击 `设置` 图标。

3. 进入 `OpenAI 兼容 API` 选项卡。

4. 点击 `添加提供者` 按钮。

5. 从下拉菜单中选择 `LMStudio`。

6. 输入 `LMStudio URL`。（默认是 `http://localhost:1234/v1`）

7. 点击 `保存` 按钮。


::: info
您不需要添加任何模型，因为 Page Assist 会自动从您配置的 LMStudio 实例中获取它们。

模型必须在 LMStudio 中加载，Page Assist 才能获取它。
:::
