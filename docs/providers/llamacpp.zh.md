# LLaMA.cpp

Page Assist 支持 LLaMA.cpp API 端点。您可以在 Page Assist 中使用任何 LLaMA.cpp API 端点。

## 添加 LLaMA.cpp API

1. 点击浏览器工具栏上的 Page Assist 图标。

2. 点击 `设置` 图标。

3. 进入 `OpenAI 兼容 API` 选项卡。

4. 点击 `添加提供者` 按钮。

5. 从下拉菜单中选择 `LLaMA.cpp`。

6. 输入 `LLaMA.cpp URL`。（默认是 `http://localhost:8080/v1`）

7. 点击 `保存` 按钮。


::: info
您不需要添加任何模型，因为 Page Assist 会自动从您配置的 LLaMA.cpp 实例中获取它们。

模型必须在 LLaMA.cpp 中加载，Page Assist 才能获取它。
:::
