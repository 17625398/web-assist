# 隐私政策

## 数据收集
Page Assist 致力于保护用户隐私，不收集任何用户数据。所有操作和数据存储都在您的浏览器本地进行。

## 浏览器权限
扩展需要以下权限才能正常运行：

- **通知**：提供系统通知
- **网站内容访问**：聊天网页功能所需
- **无限存储**：用于在浏览器本地存储聊天历史
- **活动标签页**：确保扩展在当前标签页上处于活动状态并获取截图等
- **脚本**：用于聊天网页功能
- **网络请求**：修改本地服务器的头以避免 CORS 问题

## 页面共享功能
使用页面共享功能时：

- 仅当明确连接到外部源时才会发生数据共享
- 提供自托管选项以完全控制数据
- 共享的聊天可以随时从服务器永久删除
- 删除后不保留任何数据

## 数据存储
- 所有聊天历史和设置都存储在您的浏览器本地
- 除非用户明确启动，否则不会将数据传输到外部服务器
- 用户对其数据保持完全控制

## 第三方服务
Page Assist 不与任何第三方分析或跟踪服务集成。

## 隐私政策变更
我们保留根据需要更新此隐私政策的权利。用户将收到任何重大变更的通知。
