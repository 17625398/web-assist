# Copilot

侧边栏 Copilot 是一项功能，允许您从任何网页选择文本并对其执行 AI 驱动的操作。

## 内置 Copilot 提示词

Page Assist 附带 5 个内置的 copilot 提示词：

- **总结** - 获取所选文本的简明摘要
- **改写** - 用替代词汇重写文本
- **翻译** - 将文本翻译成英语
- **解释** - 获取文本的详细解释
- **自定义** - 使用自定义提示词模板

::: warning 弃用通知
内置的 "自定义" 提示词将在未来版本中移除。请迁移到 **自定义 Copilot 提示词**（见下文）以获得更好的灵活性和多个自定义提示词。
:::

### 禁用内置提示词

您现在可以禁用任何内置的 copilot 提示词，以减少上下文菜单的混乱：

1. 进入 `设置` → `管理提示词` → `Copilot` 选项卡
2. 切换任何提示词旁边的开关以启用/禁用它
3. 禁用的提示词不会出现在上下文菜单中

## 自定义 Copilot 提示词

::: tip 新功能
您现在可以创建无限的自定义 copilot 提示词！这是创建自己的提示词的推荐方式，而不是使用内置的 "自定义" 提示词。
:::

自定义 Copilot 提示词允许您创建自己的 AI 驱动操作，当您选择文本时会出现在上下文菜单中。

### 创建自定义 Copilot 提示词

1. 进入 `设置` → `管理提示词` → `自定义 Copilot` 选项卡
2. 点击 `添加` 按钮
3. 填写表单：
   - **标题**：将出现在上下文菜单中的名称（例如，"简化文本"、"查找语法错误"）
   - **提示词模板**：您的提示词，其中 `{text}` 作为所选文本的占位符

4. 点击 `保存`

### 如何使用自定义 Copilot 提示词

1. 在任何网页上选择文本
2. 右键单击所选文本
3. 在 `Page Assist` 上下文菜单中找到您的自定义提示词
4. 点击它来处理所选文本

### 编写好的提示词

::: tip 提示词编写最佳实践
结果的质量在很大程度上取决于您如何编写提示词。请遵循以下指南：
:::

#### 1. **具体明确**
```
❌ 不好：让这个更好
✅ 好：重写以下文本，使其更专业和正式：

{text}
```

#### 2. **提供上下文**
```
❌ 不好：{text}
✅ 好：您是技术写作专家。审查以下文本并提出清晰度和简洁性的改进建议：

{text}
```

#### 3. **指定输出格式**
```
❌ 不好：检查语法：{text}
✅ 好：审查以下文本的语法和拼写错误。
列出每个错误，包括：
- 原始文本
- 更正后的文本
- 解释

文本：
{text}
```

#### 4. **使用 {text} 占位符**
始终在您的提示词中包含 `{text}`。这将被所选文本替换。

```
示例：分析以下文本的情感并将其分类为积极、消极或中性。提供您分类的理由。

文本：
{text}
```

#### 5. **必要时设置约束**
```
将以下文章总结为恰好 3 个要点。每个要点不超过 20 个单词。

文章：
{text}
```

### 示例自定义 Copilot 提示词

以下是一些您可以创建的有用的自定义提示词：

#### 语法和风格
```
标题：修复语法
提示词：审查以下文本的语法、拼写和标点错误。提供更正后的版本。

{text}
```

#### 简化
```
标题：简化文本
提示词：重写以下文本，使其能被 10 岁儿童理解。使用简单的单词和短句。

{text}
```

#### 代码审查
```
标题：审查代码
提示词：审查以下代码的：
- 潜在错误
- 性能问题
- 最佳实践
- 可读性改进

代码：
{text}
```

#### 语气调整
```
标题：使专业化
提示词：将以下文本重写为适合商业沟通的专业、正式语气。

{text}
```

#### 事实检查
```
标题：提取事实
提示词：从以下文本中提取所有事实声明。列出每个声明并注明是否需要验证。

{text}
```

### 管理自定义 Copilot 提示词

- **编辑**：点击编辑图标修改标题或提示词
- **删除**：点击删除图标删除提示词
- **启用/禁用**：切换开关以在上下文菜单中显示/隐藏提示词

更改会立即应用，无需浏览器重启！

## 如何更新内置提示词

您可以自定义默认的内置提示词：

1. 进入 `设置` → `管理提示词` → `Copilot` 选项卡
2. 点击您想要更改的提示词旁边的编辑图标
3. 修改提示词模板（必须包含 `{text}` 占位符）
4. 点击 `保存`

::: warning
编辑内置提示词会更改它们的所有未来使用。如果您想保留原始提示词，请考虑创建一个自定义 Copilot 提示词。
:::
