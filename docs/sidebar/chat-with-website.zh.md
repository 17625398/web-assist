# 侧边栏网页聊天

这是 Page Assist 的功能之一，允许您使用时尚的侧边栏从任何网页与您的 AI 模型聊天。


## 如何与网站聊天

1. 打开侧边栏
2. 启用 `与网站聊天` 选项
3. 开始与当前网站聊天


## 如何为网页聊天禁用 RAG

默认情况下，网页聊天需要有嵌入模型。您可以通过以下步骤禁用 RAG 模型：

1. 打开侧边栏
2. 点击 `设置` 图标
3. 会有一个选项 `Copilot 网页聊天设置`
4. 默认情况下，`使用向量嵌入与网站聊天` 是启用的。禁用它。
5. 您可以增加 `普通模式网站内容大小` 来增加将发送到 AI 模型的内容量。

![Disable RAG](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-19%20104323.png)

就是这样！现在您可以从任何网页与您的 AI 模型聊天，而无需使用 RAG。

## 默认启用网页聊天

您可以通过以下步骤默认启用网页聊天：

1. 打开侧边栏
2. 点击 `设置` 图标
3. 查找选项 `默认启用网页聊天（Copilot）`
