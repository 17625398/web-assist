# 侧边栏视觉（🧪）

侧边栏的视觉功能允许 LLM 查看当前网页。这类似于 [与网站聊天](/sidebar/chat-with-website.md)，但它不像 RAG。

## 如何使用具有视觉能力的 LLM 与侧边栏视觉

1. 打开侧边栏
2. 选择具有视觉能力的模型
3. 从输入框的眼睛图标启用 `视觉` 选项
4. 开始与当前网站聊天

![Vision](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-19%20104826.png)

## 如何使用不具有视觉能力的 LLM 与侧边栏视觉

Page Assist 使用 Tesseract OCR 将当前网页转换为文本。这是一个非常基本的 OCR，不是很准确，但足以了解网页的基本内容。

1. 打开侧边栏
2. 选择一个模型
3. 从输入框的眼睛图标启用 `视觉` 选项
4. 展开 `提交` 按钮下拉菜单并启用 `从图像中提取文本（OCR）`
5. 开始与当前网站聊天
