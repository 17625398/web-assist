# Page Assist 侧边栏

侧边栏是扩展的一个非常好的功能。它允许您从任何网页与您的 AI 模型聊天。您不必打开新标签并导航到网站，只需打开侧边栏并与您的 AI 模型聊天或询问有关当前页面的任何问题。


某些浏览器不支持侧边栏功能。您可以查看 [浏览器支持](/browser-support.md) 页面，了解您的浏览器是否支持。



## 如何打开侧边栏


有 3 种打开侧边栏的方法：

### 从浏览器工具栏

1. 右键单击浏览器工具栏中的 Page Assist 图标，然后选择 `打开侧边栏`。
2. 单击浏览器工具栏中的 Page Assist 图标，然后选择 `打开侧边栏`。

![Open Sidebar](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-15%20115612.png)


### 从上下文菜单

1. 右键单击任何网页，然后选择 `打开 Page Assist 侧边栏`。
![Open Sidebar](https://pub-35424b4473484be483c0afa08c69e7da.r2.dev/Screenshot%202025-02-15%20115734.png)


### 使用键盘快捷键

打开侧边栏的默认键盘快捷键是 `Ctrl+Shift+Y`。您可以在 Chrome 扩展管理页面的扩展设置中更改键盘快捷键。查看 [快捷键](/shortcuts.md) 页面了解更多信息。
